{{Cara Menjalankan Kode Web:}}
Untuk menjalankan server lokal PHP built-in (khusus Native PHP) ikuti salah satu dari kedua langkah ini:
1. gunakan perintah: "php -S localhost:8000" lewat terminal dengan cara (ctrl + `) 
2. atau pada pojok kiri atas anda bisa klik "Terminal" yang ada di sebelah kanan "Run"
